주요 기능

빨간색 기능은 직접 구현한 기능입니다.
  1. 회원관련기능
  2. 영화 관리 및 등록 기능
  3. VOD 관리 및 등록 기능
  4. 굿즈 , 스낵 관리 및 등록 기능

핵심구현기능 작동 예시


◆　메인페이지

우측 상단에 로그인 버튼과 회원가입 버튼으로 해당 기능에 접근 할 수 있습니다.
메뉴바에서 원하는 페이지로 이동할 수 있습니다.

관리자계정으로 로그인 시 관리 모드로 접근할 수 있습니다.





◆　VOD 관리

관리자계정으로 로그인 후 VOD 관리 메뉴로 접근 할 수 있습니다.
VOD 정보등록메뉴에서 원하는 영화를 검색하여 VOD로 등록하여 판매리스트에 업데이트 할 수 있습니다.(아무값도 입력하지않고 검색 시 전체결과 출력)
DB에 저장된 영화 데이터 중 현재 상영중이 아니며 VOD 리스트에 등록되지 않은 데이터만 검색결과에 출력됩니다.


등록 버튼을 누를 시 VOD 판매리스트에 등록됩니다.
VOD 삭제리스트는 VOD 리스트에 등록 되어 있는 데이터만 출력됩니다.



◆　VOD 리스트

VOD 메뉴로 접근하면 VOD 등록 된 영화 데이터가 출력 됩니다.
왼쪽 장르별 메뉴바를 통해 장르별로 리스트를 출력할 수 있습니다.
페이징 처리를 구현하였습니다.

영화 리스트 출력기능구현 코드


    @RequestMapping("vlist.do")
	public ModelAndView vlist(ModelAndView mv, @RequestParam(value = "page", required = false) Integer page, 
			@RequestParam(value = "testa", required = false) String testa) {
	
		int category = Integer.parseInt(testa);
		
		int currentPage = page != null ? page : 1;
		
		int listCount = vService.selectListCount(category);
		
		PageInfo pi = PaginationV.getPageInfo(currentPage, listCount);
		
		ArrayList<Movie> list = vService.selectList(pi ,category);
		
		if(list != null) {
			mv.addObject("list", list);
			mv.addObject("pi", pi);
			mv.addObject("cate" , category);
			mv.setViewName("vod/list");
		} else {
			throw new VodException("게시글 목록 조회에 실패하였습니다");
		}
		return mv;
		
	}
리스트 출력 DAO

    public ArrayList<Movie> selectList(PageInfo pi , int category) {
		int offset = (pi.getCurrentPage()-1) * pi.getBoardLimit();
		RowBounds rowBounds = new RowBounds(offset, pi.getBoardLimit());
		String test = "";
		if(category > 0) {
			test += category;
			return (ArrayList)sqlSession.selectList("vodMapper.selectList" , test , rowBounds );
		}else {
			
			return (ArrayList)sqlSession.selectList("vodMapper.selectList" ,test , rowBounds );	
		}
	}
리스트 출력 SQL문

    SELECT *
    FROM MOVIE
    JOIN ATTACHMENT USING(MOVIE_NO)
    WHERE MOVIE_RSTATUS = 'N'
    AND FILE_STATUS = 'Y'
    AND SORT_CODE = 20
    AND GENRE_CODE LIKE '%' || #{category} || '%'
    AND  MOVIE_VOD_STATUS = 'Y'


◆　상세페이지

VOD 리스트에서 해당 VOD를 클릭시 상세페이지로 이동합니다
VOD 상세페이지에서는 해당 VOD의 정보(스틸컷 , 예고편(유튜브) , 리뷰 등)을 확인 할 수 있습니다.
또한 내 장바구니 , 위시리스트로 접근할 수 있으며 해당 VOD를 위시리스트에 저장 , 장바구니 담기 , 구매할 수 있습니다.



◆　위시리스트

위시리스트에 저장한 VOD를 마이페이지에서 확인 할 수 있습니다.
클릭 시 해당 VOD 상세페이지로 이동하게 됩니다.



◆　장바구니

담기 버튼을 눌러 장바구니에 원하는 VOD를 담아 동시에 구매할 수 있습니다.
원하는 VOD 장바구니에서 삭제할 수 있습니다.



◆　구매하기

원하는 VOD를 구매할 수 있습니다
결제방식을 선택하여 진행할 수 있습니다.


◆　시청하기

결제한 VOD는 마이페이지 구매내역에서 VOD시청이 가능합니다.


VOD 바로보기를 통해 영상 관람이 가능합니다.

기타

  • 모든 기능은 로그인 후 진행할 수 있게끔 구현하였습니다.
